# 🐸 Pepix Project

Pepix is a limited, handcrafted pixel-art collection inscribed on the Pepecoin blockchain.  
Each Pepix blends meme culture, 8-bit artistry, and rarity-driven mechanics, with a capped supply of **4,269 Pepix**.
